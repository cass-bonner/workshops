The AWS Serverless workshop is series of Labs that intend to provide familiarity with AWS Serverless Concepts and Services, as well as provide hands-on skills to navigate the various services and build solutions on your own. The Workshop is laid out as follows: There is some Account Setup and Pre-Requisites for the users, especially if you plan to run the workshop at scale. Please see the Pre-Requisite Instructions for guidance. For those not familiar with Serverless Concepts and Terminology, please see the Serverless Overview for guidance. This page also provides more context on what we will be building so you may want to check it out regardless. From there, there are a series of Labs that need to be ran in order. We start with a simple static website, and then introduce dynamic components, add in an Amazon Lex Chatbot, and continue our journey from there. Please start your journey with the Serverless Static Website, navigating the labs either through the fix navigation bar at the top or through internal ToCs. There is also a link to the next lab at the bottom of each page.
